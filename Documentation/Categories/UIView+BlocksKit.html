Overview

Convenience on-touch methods for UIView.

Includes code by the following:

  • Kevin O'Neill. https://github.com/kevinoneill. 2011. BSD.
  • Jake Marsh. https://github.com/jakemarsh. 2011.
  • Zach Waldowski. https://github.com/zwaldowski. 2011. MIT.

UIView is only available on iOS or in a Mac app using Chameleon.

whenTouches:tapped:handler:

Abstract creation of a block-backed UITapGestureRecognizer.

- (void)whenTouches:(NSUInteger)numberOfTouches tapped:(NSUInteger)numberOfTaps handler:(BKBlock)block

Parameters

numberOfTouches

The number of fingers tapping that will trigger the block.

numberOfTaps

The number of taps required to trigger the block.

block

The handler for the UITapGestureRecognizer

Discussion

This method allows for the recognition of any arbitrary number of fingers tapping any number of times on a view. An instance of UITapGesture recognizer is allocated for the block and added to the recieving view.

This method has an additive effect. Do not call it multiple times to set-up or tear-down. The view will discard the gesture recognizer on release.
